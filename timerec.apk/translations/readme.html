Time Recording - Translations


Preparation
  • Contact developer dynamicg.android@gmail.com so that i know you're working on the translations

Setup
  1. Install this version of Time Recording: TimeRecording-ext.apk
  2. Download this file to your PC: strings.xml
  3. Translate the first few texts (e.g. "homescreenCheckinNow" and "homescreenCheckoutNow").
  4. Copy the file (strings.xml) to the top level of your phone's SD Card.
  5. Open the app. It will load the texts from the strings.xml file from step 3.

Translation
  • Same as steps 3 and 4 above: edit the strings.xml file and copy the file to your SD Card.
  • Note on newer device (e.g. Galaxy Nexus) you can attach the phone to your PC (via USB) and open/edit the file directly from the SD Card, without blocking the app.
  • Whenever you close/reopen the app, the strings.xml will get reloaded

Rollback
If you want to revert to the official version of Time Recording:
• either uninstall & reinstall it through Google Play Store
• or install the latest version from here
