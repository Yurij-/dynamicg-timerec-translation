Time Recording - Translations


Preparation
  • Contact developer dynamicg.android@gmail.com so that i know which language you're doing the translation for.


Setup
  1. Install this version of Time Recording: TimeRecording-ext.apk. This supports loading of the app's texts from the sd card.
  2. Download this file to your PC: strings.xml
  3. Alternatively, use the german version as template: strings.xml
  4. Translate the first few texts (e.g. "homescreenCheckinNow" and "homescreenCheckoutNow").
  5. Copy the file (strings.xml) to the top level of your phone's sd card.
  6. Open the app. It will load the texts from the sd card's "strings.xml" file and use within the app.


Translation
  • Same as steps 4 to 6 above: edit the "strings.xml" file and copy the file to your SD Card.
  • Note on newer device (e.g. Galaxy Nexus) you can attach the phone to your PC (via USB) and open/edit the file directly from the SD Card, without blocking the app.
  • Whenever you close/reopen the app, the "strings.xml" file will get reloaded
  • When you are done with all texts, send your version of the "strings.xml" file to me so i can integrate it into the app (zip the file so we don't lose the file encoding when sending through email)


Rollback
If you want to revert to the official version of Time Recording:
- either do a data backup and uninstall & reinstall the app through Google Play Store
- or install the latest version from here



Handling of special characters 
- \n is a newline. "a\nb" will show as "a[newLine]b"
- single and double quotes have to be escaped with backslash: \' or \"
- {1} and {2} are placeholders. "Edit \'{1}\'" may become "Edit 'Tasks'"

- \u.... are utf characters:
\u2026 = ellipsis [...]
\u2192 = right arrow [->]
\u2022 = bullet

- other special characters
&amp; = ampersand [&] 
&gt; = greater than [>]
&lt; = less than [<]
