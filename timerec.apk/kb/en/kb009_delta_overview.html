Time Recording

How can I report my overtime hours (or "time above target") over multiple months?

• Go to Menu / Worktime overview
• Set "From/To" date range, or tap [...] to pick a previously used date window
• Select a delta based "View" type. Note this is only available if you have previosuly set up either daily, weekly or monthly "target time", see Preferences on page 3.
• Use the context menu top right to hide detail lines and only show the grand total
