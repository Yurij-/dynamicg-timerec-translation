Using task 'extras'

• If not familiar with tasks start with 'How to edit tasks': KB011

• Data input (picture 1): go to the tasks edit screen and put in extra information (like project number, task tag, or similar) into the "Extra 1" and/or "Extra 2" fields

• Reporting (pictures 2 and 3):
- to find all work units matching a given "extra" use the filter panel as shown in picture 2 (i.e. the report will then include all work units that are booked to tasks that match the "extra 1" or "extra 2" filter)
- by default the "Extra 1" and "Extra 2" columns are not displayed on the reports. Use the reports' detail settings and enable those fields if required
