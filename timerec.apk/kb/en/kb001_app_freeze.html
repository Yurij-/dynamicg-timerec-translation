Time Recording Knowlege Base

The app freezes or just shows a black screen.
  • Check internal (phone) storage and make sure you're not running out of disk space. See "Storage" in the main Android System Settings. Try deleting unused apps if phone storage is full.
  • Do a backup from within the app (Preferences page 1), then uninstall and reinstall the app.
    Alternatively, update the app by installing the latest version manually. Get the APK file from here (for newer Android 4++ devices) or here (for older devices prior to Android 4 Ice Cream Sandwich)
  • Make sure the app is *not* installed on SD Card (undo "Move to SD Card" if you have done so before).
  • If using the Free version: Regularly tap the "Disable Ads" button on Preferences page 1 at the bottom - we have had a few users where loading of the ads has been blocking the app.
  • Overall, "Pro" runs more stable than "Free" (the app file is smaller and no ads are loaded). Consider migrating to Pro.
