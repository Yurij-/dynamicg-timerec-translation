Time Recording

Export and import task list
(a) Export the task list from your master device, using "Export tasks"
Note "Export to Google Drive" file will save the task list to either "TimeRecording / timerec-tasks-free.txt" or "TimeRecordingPro / timerec-tasks-pro.txt"
Also note this file is "free text" i.e. you can edit it with any text editor.
(b) After editing the text file or when setting up new devices, use "Import"




Import task list via web browser
You can trigger task import into Time Recording directly from the smartphone's web browser (using "file download"):

Preparation:
• Copy the tasks file (a) from above to a web server. Set the following file properties:
Filename = import.trtask
Mime type = application/x-timerec-tasks
• Make sure that file is accessible by the devices' web browser, then publish the given URL.

Import:
• Open the given URL in the web browser on your smartphone.
• Tap on the "file download" status bar entry

Sample URL: use "import.trtasks" on this page for testing: http://dynamicg-timerec.googlecode.com/svn/trunk/timerec.apk/test/
