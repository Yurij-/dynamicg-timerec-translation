Time Recording - Multi instance support


What is "Multi instances"?
Within the Time Recording app you can only track one individual. The app does not offer "parallel tracker" functionality.
You can however install multiple instances of the app. Each instance runs independent and stand-alone from the others, and each instance keeps its own settings and data. This is useful if you want to track multiple employees or a set of machinery.
In other words: you can install up to 20 extra Time Recording apps on a single device.


Notes
• These apps will show as «TR 01» through to «TR 20» in the app drawer or in the widget list (see "Shortcut creation" below)
• The apps will check for version updates once a week.
• You cannot install these apps through the Google Play store.
• The apps can be installed in addition to the official «Free» and «Pro» versions as available in the Google Play app store (which gives you a maximum of 22 installations per device: «Free» from Play Store, «Pro» from Play Store, plus these 20 extra instances)
• Calendar sync is unlocked and available for all app instances if Pro version is installed on the same device. Sync is supported with both 'one calendar account per app' and 'same calendar account for all apps' use cases. With the latter, sync will write parallel calendar event streams where applicable.


Backup
Back up your SD Card regularly!
All the apps write their backup data regularly onto the sd card, to folders below /sdcard/dynamicg.
Make sure you copy that folder to somewhere safe regularly.
=> Note that in case of device loss, those backups are the only means to recover data. If you lose your phone, without any external backup at hand, all data of all apps will be gone!

• Google Drive Backup:
As of version 4.19 (April/2013), "Backup to Google Drive" is enabled with all instances. Use this in addition or alternatively to regularly backing up the SD Card. Also see the "Auto backup / Google Drive" feature on Preferences (page 1, section Backup&Restore).
