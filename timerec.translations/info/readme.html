Time Recording - Translations


Preparation
  • Contact developer dynamicg.android@gmail.com so that we know which language you're doing the translation for.


Setup
  1. Get the latest "strings.xml" file from here: Resources Folder.
    When starting from scratch, use the english (values-en) or german (values-de) strings.xml file as template.
    When fixing/enhancing existing translations, use the according strings.xml file - the folder name pattern is "values-[languageCode]", e.g. values-es for spanish, values-pt for portuguese.
  2. Copy the matching strings.xml file to the top level of your SD Card. Use the "internal SD Card" if you have more than one (e.g. Samsung devices).
  3. Open the app and go to <<Menu / Preferences / Advanced>>. Click on "Translation Kit" and press "OK". Close the app.


Translation
  • Edit the "strings.xml" file. Whenever you close/reopen the app, the "strings.xml" file gets reloaded, all changes are reflected immediately.
  • Note you can edit the file directly from your PC: attach the phone via USB, open the File Explorer, go to <<Your Device>> / Internal Storage and edit the "strings.xml" file with Wordpad or any other text editor.
  • When you are done with all texts, send your version of the "strings.xml" file to me so we can integrate it into the app (zip the file so we don't lose the file encoding when sending through email)


Rollback
Go to <<Menu / Preferences / Advanced>>. Tap "Translation Kit" to disable it.


Handling of special characters 
- \n is a newline. "a\nb" will show as "a[newLine]b"
- single and double quotes have to be escaped with backslash: \' or \"
- {1} and {2} are placeholders. "Edit \'{1}\'" may become "Edit 'Tasks'"

- \u.... are utf characters:
\u2026 = ellipsis [...]
\u2192 = right arrow [->]
\u2022 = bullet

- other special characters
&amp; = ampersand [&] 
&gt; = greater than [>]
&lt; = less than [<]
